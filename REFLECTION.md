# Project Reflection

## What I built
A Smart Clinic Management System with a secure login page, admin dashboard,
patient and doctor management, appointment tracking, a REST API, Docker
containerization, and a GitHub Actions CI pipeline.

## What I learned
- How to design a relational database with foreign keys
- How JPA annotations map Java classes to database tables
- How the MVC pattern connects HTML pages to Java controllers
- How JWT tokens secure a web application
- How REST APIs work and how to build them with Spring Boot
- How Docker packages an application into a portable container
- How GitHub Actions automates building and testing on every push

## What was challenging
- Understanding the difference between @Controller and @RestController
- Wiring Spring Security with JWT for the first time
- Getting Docker Compose to connect the app container to the database container

## What I would improve next
- Add a page to create and edit patients from the browser
- Add MongoDB integration for appointment notes
- Add unit tests for the service layer
- Deploy to a cloud provider like AWS or Railway
